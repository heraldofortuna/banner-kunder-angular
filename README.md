## ¿Cómo usarlo?

#### Actualización:

Ingresa y corre en tu consola la siguiente sentencia

```html
ng update
```

#### Corre el proyecto:

Ingresa y corre en tu consola

```html
ng serve
```

#### Refencia de la API

```html
https://hack.kunderlabs.com/exam/telecom/api/products/new
```
